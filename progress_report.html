4) New Concept/Tool

One new tool/concept that we learned about in this project is MFCC or Mel Frequency Cepstral Coefficients as we mentioned before. MFCCs are commonly used in speech recognition applications and music information retrieval applications which is more relevant to us. The reason MFCCs are so useful are that they can extract key identifiers from an audio file in the form of coefficients which can then be used to evaluate/analyze that audio. The actual procedure of MFFC is the following steps taken from Wikipedia:

1. Take the Fourier transform of (a windowed excerpt of) a signal.
2. Map the powers of the spectrum obtained above onto the mel scale, using triangular overlapping windows.
3. Take the logs of the powers at each of the mel frequencies.
4. Take the discrete cosign transform of the list of mel log powers, as if it were a signal.
5. The MFCCs are the amplitudes of the resulting spectrum.

Originally we thought we would need to perform each of these steps but then we realized there are some pre-existing libraries that we can use which consolidate these actions. There also exists a matlab toolbox in the latest version of matlab which includes MFCC computations. So we will likely use some combination of the library and built in toolbox or chose one which gives us the better results.
